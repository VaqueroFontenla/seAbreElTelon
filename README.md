### Deploy GitHUb Pages

1. ```sh
   npm install -g angular-cli-ghpages@0.5.3
    ```
2. ```sh
   git checkout b gh-pages
    ```
3. ```sh
   ng build --prod --base-href /seAbreElTelon/
   ```
4. ```sh
   npx ngh --dir=dist/seAbreElTelon
   ```
5. Source GitHUb Pages: gh-pages
